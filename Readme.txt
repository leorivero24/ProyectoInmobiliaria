# Proyecto Inmobiliaria - ABM Inquilino y Propietario

## Descripción
ABM (Alta, Baja, Modificación) de **Inquilinos** y **Propietarios** desarrollado en **ASP.NET Core MVC** con **Entity Framework Core** y **MySQL**.  

---

## URLs del proyecto

- ABM Inquilinos: [http://localhost:5059/Inquilinos](http://localhost:5059/Inquilinos)  
- ABM Propietarios: [http://localhost:5059/Propietarios](http://localhost:5059/Propietarios)

---

## Ejecutar proyecto

dotnet run
